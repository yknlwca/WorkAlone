# 📖 프로젝트 소개

<div align="center">

### **💪홈트레이닝 동반자 WorkAlone**

홈트레이닝은 접근성이 뛰어나지만 지속성이 떨어지는 단점이 있습니다.
<br>
사용자가 운동을 꾸준히 할 수 있도록 강제성을 부여하며,
<br> 효율적으로 운동 습관을 관리할 수 있도록 돕습니다. </div>
